Detailed Description

A simple neural network with a fixed architecture: 3 fully connected layers, each followed by a square activation layer. It works on encrypted data: both inputs and weights, stored as CipherMatrix.

The encrypted weights of this network are either loaded from file, or encrypted from a SimpleNeuralNetPlain.
